# 📊 SQL & Database Projects Portfolio

Welcome to my SQL projects repository. 

As an Industrial Engineering & Information Systems student, I approach data from a systems perspective. I use SQL not just to query databases, but to structure information, optimize operational workflows, and extract meaningful insights from complex datasets.

This directory contains hands-on projects demonstrating my ability to design relational databases from scratch and write complex, efficient queries to solve real-world problems.

## 🎯 My Approach to Data
* **Structured Problem Solving:** Breaking down complex operational requirements into logical database schemas and queries.
* **Clean & Maintainable Code:** Writing readable SQL with clear aliases, logical formatting, and structured comments.
* **End-to-End Execution:** Handling everything from initial database architecture (DDL) to advanced data extraction and manipulation (DML).

## 🛠️ Technical Proficiencies
* **Advanced Querying:** Multi-table `JOIN` operations, Aggregations, Subqueries, `UNION`, and `VIEW` creation.
* **Database Architecture:** Relational schema design, Primary/Foreign Keys, Data Integrity constraints, and Normalization.
* **Data Engineering:** `CREATE`, `ALTER`, and `UPDATE` statements for dynamic database maintenance.
* **Ecosystems:** MySQL, Python integration (`mysql.connector`) for automated query execution.
