# 🐍 Python Data Processing Assignment

## 📌 Overview
This project was completed as part of an academic Python programming course. The assignment focuses on data processing, working with CSV files, and applying logical and structured problem-solving.

The task consists of several independent questions, each implemented in a separate Python file. The solutions involve reading structured data from CSV files, processing it according to given rules, and producing calculated results such as averages and summaries.

## 📂 Project Structure & Files
* **`01_missing_data_imputation.py`:** Reads data and performs basic initial processing (handling missing values).
* **`02_input_validation_parsing.py`:** Transforms raw user input into structured tabular data with error handling.
* **`03_conditional_matrix_logic.py`:** Applies logical conditions and filtering on randomly generated datasets.
* **`04_csv_aggregation_pipeline.py`:** Reads student grades from a CSV, calculates averages, identifies top students, and exports the data.
* **`05_oop_statistical_analyzer.py`:** Uses an Object-Oriented approach (`GradesAnalyzer` class) to clean data, calculate standard deviations, and find top performers.
* **`students.csv`:** Input data file containing student information.
* **`student_with_avg.csv`:** Output file generated by the program.

## 🛠️ Key Concepts Demonstrated
* Reading and writing CSV files using `pandas`.
* Data aggregation, averages, and standard deviations.
* Iterative and conditional logic (`try-except`, `lambda`).
* Object-Oriented Programming (OOP) in Python.
