Welcome to my C++ projects repository.

## 🎯 My Engineering Approach in C++
* **Robust Architecture:** Enforcing strict separation of concerns through modular Header (`.h`) and Implementation (`.cpp`) files.
* **Data Security (Encapsulation):** Protecting internal object states using private members, strict `const` correctness, and pass-by-reference logic.
* **Scalable Design (Polymorphism):** Building flexible systems using inheritance trees and virtual functions to allow dynamic runtime behavior.
* **Safe Memory Management:** Carefully handling dynamic memory allocation (Pointers) and ensuring leak-free execution through Virtual Destructors and Deep Copy logic (Rule of Three).
